# ICD10コードけんさく君
PHPとSQLiteを利用したICD10コードの検索サイトです。

このプロジェクトは現在[ウェブサイト](icd10kensakukun.com)にて公開中です。

---
## データベースの説明
|id|icd10_code|name|name_hankaku|大分類|中分類|similar|
|:--|:--|:--|:--|:--|:--|:--|
|ID番号|ICD10コード|疾患の名称|nameの半角表記|-|-|類似度検索処理用カラム
